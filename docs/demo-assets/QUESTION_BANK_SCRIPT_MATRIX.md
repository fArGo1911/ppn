# PPN — Question bank: inventory + recordable script matrix

> Generated from the authoritative bank (`src/demo/questionPool.ts`) via `buildBankScriptMatrix()`.
> Covers **every** seeded question, not just the demo 5 — so a compiled playlist of any mix has scripts ready.
> Readout lines are category-led and order-agnostic; when compiled into a playlist they gain a running
> "Question N" number, and answers gain a rotated answer-review lead-in (see `/config#brand-media`).
> **No audio is generated by the app** — these are scripts to record (e.g. in ElevenLabs) and drop in as MP3s.
